# Contributing to the ABAP Test Isolation Demos

:+1::tada: First off, thanks for taking the time to contribute! :tada::+1:

The following is a set of guidelines for contributing to this repository. These are mostly guidelines, not rules. Use your best judgment, and feel free to propose changes to this document in a pull request.

## Why should you read this guide?

Following these guidelines helps to communicate that you respect the time of the developer managing and developing this open source project. 
In return, they should reciprocate that respect in addressing your issue, assessing changes, and helping you finalize your pull requests.

And: It's pretty short.

## What kind of contributions am I looking for?
I'm looking for any kind of contribution, really. Questions, formatting, discussions, new tools (or existing ones I missed), anything. 

## Your First Contribution
Just create an pull request with whatever contribution you deem helpful and I'll do my best to look at it as quickly as possible.
